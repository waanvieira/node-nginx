Docker com NodeJS, MYSQL e Nginx como proxy reverso

Como rodar o projeto

- Clone o projeto em uma pasta
- Ir até a pasta que clonou o projeto
- Atualmente verificar se esta na branch master se não estiver fazer o checkou na branch (checkout master)
- executar o comando docker-compose up -d
- Abrir o navegador no endereço http://localhost ou http://127.0.0.1
